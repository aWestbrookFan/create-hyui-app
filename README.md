#### 脚手架价值：为了加速项目创建和代码规范，选取了电脑端、手机端的vue模板、jq网站等，
#### 实现：通过对远程仓库的拉取来进行快速创建项目。

#### 1. 脚手架全局安装

```
npm i create-hyui-app -g
```

#### 2. 脚手架安装完成后，直接使用终端打开输入
```
create-hyui-app
```
